Data Science Projects

These projects showcase ML and DL techniques on large datasets from Kaggle and Hugging Face, utilizing Python libraries like pandas, scikit-learn, and PyTorch. They cover data collection, preprocessing, and models ranging from simple regression to neural networks and transformers.

S2TP Automatic Speech Recognition (ASR)

This project explores Automatic Speech Recognition (ASR) technology with a personal focus on its potential impact for the deaf community. I utilized the LibriSpeech dataset, which contains over 1,000 hours of English speech, to develop a Connectionist Temporal Classification (CTC)-based model. The model was trained to transcribe audio into text by predicting character sequences, addressing challenges like variable speech rates and background noise. I also conducted a comparative analysis with Meta's Wav2Vec model, a state-of-the-art self-supervised learning approach, to benchmark performance. Despite encountering issues with model convergence and accuracy (achieving a word error rate of approximately 25%), this project provided deep insights into the complexities of ASR systems, including the need for robust preprocessing, large-scale data, and computational resources. Future improvements could involve fine-tuning with domain-specific data or exploring hybrid architectures.

Sentiment Analysis for Movie Reviews

This project implements natural language processing (NLP) techniques to classify sentiment in a dataset of 50,000 IMDB movie reviews. I began with data preprocessing steps, including tokenization, stop-word removal, and lemmatization, to prepare the text for modeling. Two approaches were compared: a Long Short-Term Memory (LSTM) network, which captures sequential dependencies in text, and a BERT transformer, leveraging contextual embeddings. The LSTM model was built using PyTorch with an embedding layer, bidirectional LSTM, and a dense output layer, achieving an accuracy of 89%. The BERT model, fine-tuned using Hugging Face's Transformers library, outperformed it with a 93% accuracy, thanks to its ability to understand bidirectional context. Performance analysis included confusion matrices, precision-recall curves, and error inspection, revealing BERT's strength with nuanced reviews. This project highlighted the trade-offs between model complexity, training time, and predictive power in NLP tasks.

Housing Price Prediction Model

This project applies machine learning to predict housing prices using the California housing dataset, which includes features like median income, house age, and location coordinates. I performed extensive feature engineering, such as creating interaction terms (e.g., income × rooms) and normalizing data to handle scale differences. Three regression models were compared: Linear Regression as a baseline, Random Forest for non-linear relationships, and Gradient Boosting (using XGBoost) for enhanced predictive power. I implemented 5-fold cross-validation to ensure robust evaluation, with metrics including R², mean absolute error (MAE), and root mean squared error (RMSE). The Gradient Boosting model excelled, achieving an R² of 0.89 and an MAE of $18,500 on the test set, outperforming Linear Regression (R² 0.65) and Random Forest (R² 0.82). Detailed analysis revealed the importance of location and income features, with potential improvements including hyperparameter tuning or incorporating external data like interest rates.
